# Learn Bash Shell

A collection of Bash scripts for learning and practicing shell scripting.

## Prerequisites
- Linux / macOS terminal (or WSL on Windows)
- Bash installed (usually pre-installed on most systems)

## Running the scripts
1. Give execute permission:
   ```bash
   chmod +x script.sh```
2. Run the script:
   ```bash
   ./script.sh```
